# 关于Web项目自定义开发
1. 下载node-v12.4.0-linux-x64
    - windows下请模拟配置
2. `cd web`
3. `export PATH=$PATH:node-v12.4.0-linux-x64/bin`
4. `npm config set registry https://registry.npm.taobao.org`
5. `npm install`
6. `npm run build`

## 若要进入开发模式
1. `cd web`
2. `bash run_dev.sh`
